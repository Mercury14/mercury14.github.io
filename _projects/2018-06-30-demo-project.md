## Current Personal Productivity System

Like many people, I've found a need to tweak my personal productivity system as a result of the increased time that I am spending working from home. I work on a number of concurrent work projects and initiatives at any one time, so I need to manage my attention well if I am going to get anything meaningful done.

### Daily System

I then use the following tools and approaches to support my daily activities:

Emails (Outlook)
* I process my emails at 3 x set intervals during the day. 
* Emails that are noting or info are deleted
* Emails requiring action 
* If I can do the task in a few minutes and I have the time, then I just do the task
* Otherwise I flag the email for further action 

Tasks (MS Planner)
* Flagged tasks are automatically created in MS Planner using this <a href ="https://emea.flow.microsoft.com/en-us/galleries/public/templates/67477f30373611e7870df906aa521b7a/create-planner-tasks-for-flagged-emails-in-office-365/">Power Automate flow</a>
* Any actions that I pick up during the day are put into planner
* I set due dates and filter by this view to understand what tasks need to be completed today
